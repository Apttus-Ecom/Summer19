Public addProductToCart
addProductToCart(product: Product, quantity: number, merge: boolean, attributeValues?: Array, productOptions?: Array, quick: boolean, rollBackOnFail: boolean, timeout: number)

Adding a Product to Cart is the most common Cart action. Will trigger a reprice operation asynchronously.

Example:

      import { CartService, Product, ProductAttributeValue } from '@apttus/ecommerce';

      export class MyComponent implements OnInit{

           constructor(private cartService: CartService){}

           addProductToCart(product: Product, quantity: number, productAttributeValueList: Array<ProductAttributeValue>): void{
               this.cartService.addProductToCart(product, quantity, productAttributeValueList, null, true, true).subscribe(
                   () => {...},
                   err => {...}
               )
           }
      }
Parameters :
Name Type Optional Default value Description
product Product No

Product object for method
quantity number No 1

number value for quantity
merge boolean No true

true/false on whether to merge this product with the same product already in the cart
attributeValues Array<ProductAttributeValue> Yes

an array of product attribute values associated with the product
productOptions Array<ProductOptionForm> Yes

an array of product option form instances associated with the prodcut
quick boolean No true

(deprecated) A boolean flag that can be set to true if you wish to wait for the repricing cart to finish before returning
rollBackOnFail boolean No true

(deprecated) a boolean flag that will delete the item from the cart if the cart fails to reprice
timeout number No 60000

(deprecated) The maximum amount of time the reprice operation should take to complete
Returns : Observable<void>

a void observable when the operation is complete
Public bulkAddProductToCart
bulkAddProductToCart(productFormList: Array, cart?: Cart)

Method adds multiple products to the cart specified in an array mapping a product code to a quantity. If a product code is not available in the catalog, the method will throw an error specifying the product codes that cannot be added.

Example:

      import { CartService } from '@apttus/ecommerce';

      export class MyComponent{

           constructor(private cartService: CartService){}

           addMultipleProductsToCart(){
               const productList: Array<CartProductForm> = [
                   {productCode: '1111', quantity: 3},
                   {productCode: '1112', quantity: 5}
               ];
               this.cartService.bulkAddProductToCart(productList).subscribe(productCodes => {...});
           }
      }
Parameters :
Name Type Optional Description
productFormList Array<CartProductForm> No

an array of CartProductForm instances with the product and quantity data of the items
cart Cart Yes

the cart record to add the products to. If undefined, products will be added to the current cart
Returns : Observable<Array<string>>

an Array of strings in a cold observable of the product codes that were added to the cart

Public filter
filter()

The global filter used for retrieving carts. Will return only those carts where the user is the owner and matches the current selected account. Also further filters the carts returned by the currently selected price list and effective price list

Returns : Observable<string>

an Observable of type string with the filter query
Public getMyCart
getMyCart()

The primary method to get the user's current cart. This is a hot observable that will be updated when the state of the cart changes.

Example:

      import { CartService } from '@apttus/ecommerce';

      export class MyComponent implements OnInit{
           cart: Cart;
           cart$: Observable<Cart>; // can be used in the template with {{(cart$ | async)?.Name}}

           constructor(private cartService: CartService){}

           ngOnInit(){
               this.cartService.getMyCart().subscribe(cart => this.cart = cart);
               // or
               this.cart$ = this.cartService.getMyCart();
           }
      }
Returns : Observable<Cart>

A hot observable containing the single cart instance

Public priceCart
priceCart(cart: Cart, cartItemList: Array, timeout: number, rollBackOnFail: boolean, force: boolean, attempts: number)

priceCart method triggers the server-side pricing engine for the cart

Example:

      export class MyComponent implements OnInit{

           constructor(private cartService: CartService){}

           repriceCart(cart: Cart): void{
               this.cartService.priceCart(cart, null, true, true).subscribe(
                   () => {...},
                   err => {...}
               );
           }
      }
Parameters :
Name Type Optional Default value Description
cart Cart No

the cart record to reprice
cartItemList Array<CartItem> No null

(deprecated) an array of cart items that will be deleted if the reprice fails
timeout number No 30000

(deprecated) the maximum time to wait for the reprice operation. Defaults to 30 seconds
rollBackOnFail boolean No true

(deprecated) when set to true, will delete the corresponding cart item if the reprice fails
force boolean No false

sets all line items to 'pending' before repricing
attempts number No 100

the maximum number of times to poll the reprice operation. Increase this number if done on a large cart that will take a while to reprice.
Returns : Observable<void>

a void observable when the operation completes
