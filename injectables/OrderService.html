Public convertCartToOrder
convertCartToOrder(order: Order, primaryContact: Contact, cart?: Cart)

Method converts the current cart into an order record. Will set the cart status to 'Finalized'.

Example:

      import { OrderService, Order, Contact, Cart, ContactService } from '@apttus/ecommerce';
      import { Observable } from 'rxjs/Observable';

      export class MyComponent implements OnInit{
           order: Order;

           constructor(private orderService: OrderService, private contactService: ContactService){}

           convertCartToOrder(){
               // Use the contact record for the current user
               this.contactService.getMyContact().take(1).subscribe(contact => {
                    const order: Order = new Order();
                    order.Name = 'Sample Order';
                    this.orderService.convertCartToOrder(order, contact).subscribe(o => this.order = order);
               });
           }
      }
Parameters :
Name Type Optional Default value Description
order Order No new Order()

an optional instance of an order to predefine values on the final order record that gets created
primaryContact Contact No

the primary contact to assign to the order
cart Cart Yes

the cart record to convert. Will use the current active cart if left undefined.
Returns : Observable<Order>

An observable of the order instance that gets created

Public getMyOrders
getMyOrders(accountId: string, days?: number, limit: number, pageNumber: number, orderBy: string, orderDirection: "ASC" | "DESC")

method returns a list of orders for the current user

Example:

      import { OrderService, Order } from '@apttus/ecommerce';
      import { Observable } from 'rxjs/Observable';

      export class MyComponent implements OnInit{
           orderList: Array<Order>;
           orderList$: Observable<Array<Order>>;

           constructor(private orderService: OrderService){}

           ngOnInit(){
                this.orderService.getMyOrders().subscribe(o => this.orderList = o);
                // or
                this.orderList$ = this.orderService.getMyOrders();
           }
      }
Parameters :

Public mergeCartItems
mergeCartItems(orderItemList: Array, cart?: Cart)

Method is used for adding an existing order to a cart (i.e. reorder). If the items already exist in the cart, it will increment the quantity.

Example:

      import { OrderService, Order, Cart } from '@apttus/ecommerce';
      import { Observable } from 'rxjs/Observable';

      export class MyComponent implements OnInit{
           constructor(private orderService: OrderService){}

           addOrderItemsToCart(order: Order){
               this.orderService.mergeCartItems(order.OrderLineItems)
                .subscribe(
                    () => {...},
                    err => {...}
                );
           }
      }
Parameters :
Name Type Optional Description
orderItemList Array<OrderLineItem> No

a list of order items to merge
cart Cart Yes

the cart record to add the items to. If left undefined or null, will use the current cart.
Returns : Observable<void>

a void observable when the operation completes
