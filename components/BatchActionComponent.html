Description

Batch action component is used to display the list of currently active batch actions as buttons on the UI.

Preview

Usage

```typescript import { BatchActionsModule } from '@apttus/elements';

@NgModule({ imports: [BatchActionsModule, ...] }) export class AppModule {} ```

<div class="d-flex flex-md-column align-items-center">
    <ng-container
      *ngFor="let action of actions$ | async"
      >
      <div
        *ngIf="action.isDisabled() && action.isVisible()"
        class="tooltipWrapper w-100"
        [tooltip]="action.getTooltipText()"
        container="body"
      >
        <button
          class="d-flex align-items-center btn btn-link my-md-1 border"
          (click)="handleClick(action)"
          [disabled]="action.isDisabled()"
        >
          {{action.getDisplayName()}}
        </button>
      </div>
  
  
      <button
        *ngIf="!action.isDisabled()"
        class="d-flex align-items-center btn btn-outline-primary my-md-1"
        (click)="handleClick(action)"
      >
        {{action.getDisplayName()}}
      </button>
    </ng-container>
  </div>
