Description

The payment method layout component is a listing component for payment method object It is used to display all available payment method for logged in user and will select add credit/debit card option if no payment method exist.

import { PaymentComponentModule } from '../payment/payment.module';

@NgModule({
  imports: [PaymentComponentModule, ...]
})
export class AppModule {}

OnInit OnDestroy

Example

<apt-payment-method

   (isAnyMethodExist)='selectDefaultPaymentOption($event)'
></apt-payment-method>
