Description

Mini profile allows user to login to the website through their existing ID's. For the new user, there is an option to register to the site . If the user has forgotten the password there is an option to reset password.

Usage

import { MiniProfileModule } from '@apttus/elements';

@NgModule({
  imports: [MiniProfileModule, ...]
})
export class AppModule {}

Implements

OnInit

Example

<apt-mini-profile
   (onRegister)="handleOnRegister($event)"
   (onLogin)="handleOnLogin($event)"
></apt-mini-profile>
