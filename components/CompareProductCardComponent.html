projects/apttus/elements/src/lib/product-card/compare-product-card/compare-product-card.component.ts

Description

Compare product card is used as the header in the compare table. Compare cards show product details including images, price, short description, and add to cart button.

Preview

Usage

import { ProductCardModule } from '@apttus/elements';

@NgModule({
  imports: [ProductCardModule, ...]
})
export class AppModule {}

Extends

ProductCardComponent

Example

<apt-compare-product-card [product]="product"></apt-compare-product-card>
