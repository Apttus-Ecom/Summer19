projects/apttus/elements/src/lib/product-card/compare-product-card/compare-product-card.component.ts

Description

Compare product card is used as the header in the compare table. Compare cards show product details including images, price, short description, and add to cart button.

Preview

Usage

import { ProductCardModule } from '@apttus/elements';

@NgModule({
  imports: [ProductCardModule, ...]
})
export class AppModule {}

Extends

ProductCardComponent

Example

<apt-compare-product-card [product]="product"></apt-compare-product-card>

Inputs

canAddToCart
Type : boolean
Default value : true

Flag to show the add to cart button on the card.
product
Type : Product | string

The product associated with this component.
showCategory
Type : boolean
Default value : true

Flag to show the category of the product.
showCode
Type : boolean
Default value : true

Flag to show the code of the product.
showPrice
Type : boolean
Default value : true

Flag to show the price of the product.
thumbnail
Type : boolean
Default value : true

Flag to show the thumbnail image on the product card.
type
Type : "card" | "media"

Display type of the product card.
