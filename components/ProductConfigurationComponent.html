Description

Product configuration component is used to show the hierarchical view of the configurations for a given product or cart line item.

Usage

import { ProductConfigurationModule } from '@apttus/elements';

@NgModule({
  imports: [ProductConfigurationModule, ...]
})
export class AppModule {}

Implements

OnChanges OnDestroy OnInit

Example

<apt-product-configuration
   [productId]="product"
   [cartItemId]="lineItem"
   [root]="isRoot"
   [optionFormList]="productOptionForms"
   [attributeValue]="productAttributeValue"
   (configurationChange)="handleConfigurationChange($event)"
></apt-product-configuration>
